# 钉钉字段捷径 - 物流查询助手

一个基于钉钉AI表格字段捷径的物流查询工具，可以快速查询快递物流信息并返回详细的物流状态。

## 功能特性

- 🚚 **智能物流查询**：支持通过物流单号查询快递状态
- 🌍 **多语言支持**：支持中文、英文、日文界面
- 📦 **快递公司识别**：可指定快递公司或自动识别
- 📊 **详细物流信息**：返回最新状态、状态描述、更新时间等详细信息
- 🔗 **钉钉集成**：无缝集成到钉钉AI表格中

## 输入参数

- **物流单号**（必填）：需要查询的快递单号
- **快递公司**（可选）：快递公司名称，如不填写将自动识别

## 输出结果

查询成功后将返回以下信息：

- **最新状态**：当前快递状态（如：已签收、运输中等）
- **状态描述**：详细的状态描述信息
- **快递更新时间**：最后更新时间
- **快递公司**：识别出的快递公司
- **物流详情**：完整的物流轨迹信息
- **数据写入时间**：数据记录时间

## 技术架构

- **框架**：基于 `dingtalk-docs-cool-app` 开发
- **语言**：TypeScript
- **API**：集成石榴AI物流查询服务
- **授权**：使用石榴AI API Key进行身份验证

## 配置说明

### API配置

项目使用石榴AI的物流查询服务，需要在 `config.json` 中配置API Key：

```json
{
    "authorizations": {
        "shiliuAIApiKey": "your-api-key-here"
    }
}
```

### 域名配置

当前配置的API服务域名为：`13a54707.r1.cpolar.top`

## 使用说明

1. 在钉钉AI表格中安装此字段捷径
2. 配置石榴AI API Key
3. 输入物流单号（必填）和快递公司（可选）
4. 系统将自动查询并返回详细的物流信息

## 注意事项

- 请确保API Key有效且有足够的查询额度
- 物流单号格式需要正确，否则查询可能失败
- 建议填写快递公司名称以提高查询准确性
